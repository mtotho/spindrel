# Marp Slides

Agent tool for creating presentation decks from Markdown.

Built on [Marp](https://marp.app) (`@marp-team/marp-cli`), an open-source Markdown presentation ecosystem by the [Marp team](https://github.com/marp-team). Marp is MIT licensed.

## Requirements

- Node.js / `npx`. The integration can use `npx --yes @marp-team/marp-cli`, or the admin dependency installer can install `@marp-team/marp-cli`.
- Chromium / Chrome for PDF and PPTX export.

## What's included

- `tools/marp_slides.py` — `create_marp_slides` tool (Markdown to HTML/PDF/PPTX)
- `skills/marp_slides.md` — Marp syntax reference and workflow guide
