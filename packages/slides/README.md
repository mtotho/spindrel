# Slide Decks

Agent tool for creating presentations from Markdown.

Built on [Marp](https://marp.app) (`@marp-team/marp-cli`) — an open-source Markdown presentation ecosystem by the [Marp team](https://github.com/marp-team). Marp is MIT licensed.

## Requirements

- Node.js / npx (Marp CLI is auto-installed via `npx` on first use, or install globally: `npm install -g @marp-team/marp-cli`)

## What's included

- `tools/slides.py` — `create_slides` tool (Markdown → HTML/PDF/PPTX)
- `skills/slides.md` — Marp syntax reference and workflow guide
