Configuration & Scope

Scope levels
session — this bot in this channel (most specific)
bot — this bot across all channels
channel — any bot in this channel
global — any bot in any channel
Retrieval is inclusive: a bot always sees its own scope + broader scopes (bot, channel, global).
Bot YAML (bot-scoped)
filesystem_indexes:
  - root: /path/to/project
    patterns: ["**/*.py", "**/*.md"]
    cooldown_seconds: 120
    watch: true
YAML indexes are always bot scope (cross-channel).
Agent tools: index_directory_now(path, scope), search_codebase(query), delete_filesystem_index(path, scope), list_indexed_directories()
