Slash Commands

/ask <bot-id> <message>Route a message to a specific bot. Bot responds in the channel attributed to its display name.
/bot [id]Switch default bot for this channel (used for @mentions). Session stays the same.
/botsList all available bots and their IDs.
/contextShow context breakdown for this channel's session.
/plan [...]Manage plans in this channel's session.
/compactForce-compact the session context.

Note: Sessions are now channel-scoped. One session per channel, shared across all bots. Switching bots with /bot keeps the same session.

@-tagging: type @bot-id or @skill-name as plain text in any message — the agent resolves them automatically. No Slack autocomplete for internal IDs; use /bot for switching.
