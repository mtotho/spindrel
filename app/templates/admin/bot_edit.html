Features

Persona injects a persistent personality/tone file as a separate system message (distinct from the system prompt — edit it here when enabled). Context Compaction auto-summarizes old turns so the context window never fills up.

Injected as a [PERSONA] system message. Separate from the system prompt — use this for personality/tone.

Memory

Short facts stored between turns. At compaction the bot decides what to save (via save_memory tool) using the compaction prompt. On each new turn, relevant memories are retrieved by semantic similarity and injected into context. Cross-session shares all memories across conversations; otherwise scoped to the current session.
